Об авторе

Привет! Меня зовут Костя Миков и я начинающий веб-разработчик. По основному образованию я инженер-строитель, но, решив идти в своей жизни "не по течению", я переквалифицировался на более близкую для меня и актуальную в современном мире профессию. Теперь я строю API и веб-страницы разной сложности :)

Начал я с курса Яндекс.Практикума под кодовым названием "Веб-разработчик". Девять месяцев моей профильной учебы отражены в этом самостоятельном проекте. Здесь использованы: сервер на Node.js (express) в связке с базой данных MongoDB, сборка проекта Webpack'ом, вёрстка по BEM и, конечно, HTML5, CSS3 и нативный JavaScript.
